# Air Conditioner

云空调，便携小空调，为你的夏日带去清凉！

## Features

### 优势

- 随时随地打开空调
- 便携
- 低功耗
- 静音
- 操作简单
- 安装便捷

### 劣势

- 没有风

## Dev

```bash
# yarn dev
yarn start
# http://localhost:3000/

yarn build
# ./build
```
